Detailed Description

Concrete strategy for low-frequency, minimal watering.

Implements a watering strategy with low water amounts and infrequent watering schedules, suitable for drought-tolerant plants like succulents.

System Role: Strategy for drought-tolerant plants (Succulents, some Herbs). Applied via Builder during plant construction. Executed by WaterPlantCommand during care. Shared via flyweight factory across all low-water plants.

Pattern Role: Concrete Strategy (implements low-water algorithm)

Related Patterns:

  • WaterStrategy: Implements abstract interface
  • LivingPlant: Context using this strategy
  • Builder: Assigns to appropriate plant types
  • Singleton: Cached by flyweight factory

System Interactions:

  • Builder assigns to drought-tolerant plant species
  • water() applies minimal water amount directly to plant object
  • getID() returns consistent identifier for caching
  • Shared across all succulent instances
